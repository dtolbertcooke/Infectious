How to use this application

Registration

Register for the site under the login. This information will be displayed on your profile page.

Login

Log in using credentials made once registered. You will then be directed to your profile page. From here you can edit your profile, use the recommendation system or log out once you are done using the application.

Edit profile

After clicking 'Edit profile' on your profile page, you can change the information you entered when you registered. Once changes have been submitted, you can view the changes on your profile page.

Recommendation

This is the main feature of the application. From here, you can find movies similar to the movie entered so long as the movie entered is in the dataset.
